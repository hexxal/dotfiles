# Ansible dotfiles configuration (Fedora)

- Install git
  - `sudo dnf install git`
- Install ansible
  - `sudo dnf install ansible`
- Install ansible playbook(s)
  - `ansible-playbook -i hosts -K playbook-cli.yml`
  - `ansible-playbook -i hosts -K playbook-gui.yml`
